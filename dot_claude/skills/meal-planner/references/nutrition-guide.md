## 1. Repères Nutritionnels PNNS

### Aliments à augmenter

| Groupe alimentaire          | Recommandation                                                            | Source    |
| --------------------------- | ------------------------------------------------------------------------- | --------- |
| Fruits et légumes           | Au moins 5 portions/jour (80-100 g par portion)                           | PNNS, OMS |
| Légumineuses                | Au moins 2 fois/semaine (lentilles, pois chiches, haricots)              | PNNS      |
| Fruits à coque non salés    | Une petite poignée par jour (noix, amandes, noisettes)                   | PNNS      |
| Féculents complets          | Privilégier les versions complètes ou semi-complètes                     | PNNS      |
| Poisson                     | 2 fois/semaine dont 1 poisson gras (saumon, sardines, maquereau)         | PNNS      |
| Huiles végétales            | Privilégier olive, colza, noix pour cuisson et assaisonnement            | PNNS      |
| Produits laitiers           | 2 portions/jour                                                           | PNNS      |
| Fibres alimentaires         | Minimum 25 g/jour                                                         | OMS       |
| Fait maison                 | Privilégier la cuisine maison (contrôle des ingrédients)                 | PNNS      |
| Produits de saison et locaux | Préférer les produits de saison et locaux                                | PNNS      |

### Aliments à réduire ou limiter

| Aliment / substance             | Limite                                                                  | Source     |
| ------------------------------- | ----------------------------------------------------------------------- | ---------- |
| Charcuterie                     | Max 150 g/semaine ; préférer jambon blanc / de volaille                 | PNNS       |
| Viande rouge                    | Max 500 g/semaine (porc, bœuf, veau, agneau)                           | PNNS       |
| Sel                             | < 5 g/jour (= < 2 g sodium/jour)                                       | PNNS, OMS  |
| Sucres libres                   | < 10 % de l'apport énergétique (~50 g pour 2000 kcal) ; idéalement < 5 % | OMS      |
| Graisses saturées               | < 10 % de l'apport énergétique                                         | OMS        |
| Graisses trans industrielles    | Éliminer totalement                                                     | OMS        |
| Boissons sucrées                | Réduire autant que possible                                             | PNNS, OMS  |
| Produits ultra-transformés      | Réduire ; préférer Nutri-Score A/B                                      | PNNS       |
| Édulcorants non sucrés          | Ne pas utiliser comme substitut du sucre                                | OMS        |
| Alcool                          | Max 2 verres/jour, pas tous les jours ; max 10 verres/semaine           | PNNS       |

## 2. Principes de Perte de Poids Saine

| Principe                                         | Détail                                                                                              | Source    |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------- | --------- |
| Objectif réaliste                                | 5-10 % du poids initial = perte cliniquement significative                                          | HAS       |
| Pas de régime restrictif                         | Rééquilibrage alimentaire durable, pas de régime yo-yo                                              | HAS, PNNS |
| Approche de première intention                   | Alimentation équilibrée + activité physique + réduction de la sédentarité                           | HAS       |
| Contrôle des lipides                             | Lipides totaux ≤ 30 % de l'énergie pour prévenir la prise de poids                                 | OMS       |
| Suivi pondéral                                   | Pesée hebdomadaire pour suivre la tendance                                                          | HAS       |
| Éviter les produits « minceur »                  | Pas de produits marketing sans preuve d'efficacité                                                  | PNNS      |
| Encadrement médical si régime très restrictif    | Ne pas descendre sous 1200 kcal/jour sans suivi médical                                             | HAS       |

## 3. Équilibre des Repas

### Répartition des macronutriments (OMS)

| Macronutriment | Part de l'apport énergétique | Notes                                                     |
| -------------- | ---------------------------- | --------------------------------------------------------- |
| Glucides       | 45-75 %                      | Privilégier glucides complexes et complets                |
| Protéines      | 10-15 %                      | Peut dépasser 15 % si activité musculaire régulière       |
| Lipides        | 15-30 %                      | Saturés < 10 %, trans < 1 % (industriels : 0 %)          |

### Composition d'une assiette équilibrée

- **½ assiette** : légumes (cuits ou crus)
- **¼ assiette** : protéine (viande maigre, poisson, œufs, légumineuses)
- **¼ assiette** : féculent complet (riz complet, pâtes complètes, quinoa, pommes de terre)
- **Assaisonnement** : huile végétale (olive, colza), herbes, épices, citron
- **Cuisson** : privilégier vapeur, four, poêle avec peu de matière grasse (éviter fritures)

## 4. Hydratation et Boissons

| Recommandation                                                              | Source |
| --------------------------------------------------------------------------- | ------ |
| L'eau est la seule boisson recommandée                                      | PNNS   |
| Réduire les boissons sucrées (sodas, jus de fruits industriels)             | PNNS   |
| Les jus de fruits, même 100 % pur jus, ne comptent pas comme un fruit      | PNNS   |
| Ne pas remplacer le sucre par des édulcorants                               | OMS    |
| Potassium : au moins 3510 mg/jour, via fruits et légumes frais             | OMS    |

## 5. Activité Physique et Alimentation

| Recommandation                                                                        | Source     |
| ------------------------------------------------------------------------------------- | ---------- |
| Minimum 30 min d'activité modérée/jour (150-300 min/semaine)                          | PNNS       |
| Ou 75-150 min/semaine d'activité intense                                              | PNNS       |
| Renforcement musculaire au moins 2 fois/semaine                                       | PNNS       |
| Réduire le temps sédentaire ; interrompre les longues périodes assises                | PNNS       |
| Protéines : peuvent dépasser 15 % de l'énergie si activité de renforcement musculaire | OMS        |
| Glucides : besoins accrus les jours d'activité intense                                | OMS        |
| Vitamine D : 15 µg/jour ; attention si entraînement en intérieur                     | ANSES      |
| Fer : besoins potentiellement accrus si activité physique intense                     | ANSES      |

## Rappels pour la Génération de Menus

- Équilibrer les macros sur la semaine, pas sur chaque repas individuel
- Varier les sources de protéines (volaille, poisson, œufs, légumineuses, bœuf/porc en quantité limitée)
- Respecter les limites hebdomadaires : charcuterie < 150 g, viande rouge < 500 g
- Intégrer au moins 2 repas végétariens par semaine
- Inclure 2 portions de poisson par semaine dont 1 gras
- Privilégier les légumes de saison
- Favoriser les cuissons douces (vapeur, four, poêle)
- Les fruits (raisin, clémentines, poires) constituent les collations si besoin
