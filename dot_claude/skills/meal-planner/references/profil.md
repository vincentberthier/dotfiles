**Petit-déjeuner (composition fixe, non planifié)**

| Aliment               | Quantité | Kcal estimées |
| --------------------- | -------- | ------------- |
| Lait chocolaté        | 250 ml   | ~155 kcal     |
| Œufs                  | 2        | ~160 kcal     |

- Répartir les macros sur la durée du plan, pas au repas près.
- Jour de sport : augmenter glucides et protéines de +15-20%.
- Privilégier : protéines maigres, légumes de saison, féculents complets.
- Déjeuners légers : salades composées, soupes, tartines, etc.
- Dîners plus consistants mais raisonnables.

# Contraintes Pratiques

| Contrainte                     | Valeur                                                                  |
| ------------------------------ | ----------------------------------------------------------------------- |
| Temps de préparation moyen     | 15-20 min                                                               |
| Temps de préparation max       | 30 min (exceptionnel)                                                   |
| Batch cooking / meal prep      | Non                                                                     |
| Capacité congélation           | Faible (quelques portions max)                                          |
| Fréquence courses principales  | Toutes les 2 semaines, livraison en ligne (secs, conserves, surgelés, viande) |
| Complément frais               | Course à pied au supermarché en milieu de cycle (sac à dos, pas de voiture)   |
| Lieu des repas                 | Domicile                                                                |

- Prévoir des quantités facilement partitionnables, consommées entièrement sur la période de 2 semaines.
- Exception : épices, bouillon, huile, moutarde, etc. (fonds de placard).
